## Apa itu Go atau Golang ?

[Go](https://go.dev) adalah bahasa pemrograman yang dikembangkan oleh Google.
Ini bertujuan untuk menyediakan kecepatan pengembangan, efisiensi, dan skalabilitas.
Go menawarkan fitur seperti garbage collection, concurrency,
dan bahasa yang sederhana dan mudah digunakan.

## Mengapa Harus Golang ?

Ada beberapa alasan mengapa seseorang mungkin memilih Go:

1. Kecepatan kompilasi dan eksekusi yang cepat
2. Mendukung concurrency dengan baik melalui goroutine dan channel
3. Garbage Collection otomatis
4. Kode yang bersih dan mudah dipahami
5. Dukungan yang kuat dari komunitas dan perusahaan-perusahaan besar seperti Google, Uber, Dropbox
6. Bisa digunakan untuk aplikasi web, backend, sistem, dan aplikasi cloud.

Concurrency :

> Concurrency adalah kemampuan sistem untuk menjalankan beberapa tugas secara bersamaan atau seolah-olah bersamaan.
> Dalam pemrograman, concurrency digunakan untuk memecahkan masalah besar menjadi bagian-bagian yang lebih kecil
> dan menjalankan setiap bagian secara paralel sehingga dapat meningkatkan efisiensi dan performa.
>
> Concurrency dapat diterapkan pada banyak tingkatan, mulai dari aplikasi multithreading hingga sistem distribusi.

## Seperti apa golang ?

Go adalah bahasa pemrograman procedural dengan fitur-fitur seperti OOP (Object Oriented Programming).
Meskipun Go tidak memiliki konsep class dan inheritance seperti Java, Go memiliki fitur seperti structs,
interfaces, dan methods untuk membangun aplikasi yang terstruktur dan terorganisir.

Go juga menekankan pada konsep concurrency dan memiliki fitur khusus untuk menangani concurrency seperti
goroutine dan channel, yang membuat Go cocok untuk membangun aplikasi yang membutuhkan kinerja dan efisiensi tinggi.

## Bagaimana install Golang di windows ?

Untuk menjalankan Go di Windows, ikuti langkah-langkah berikut:

1. Download dan install Go dari situs resmi https://go.dev/dl/
2. Setelah instalasi selesai, buka Command Prompt atau PowerShell dan cek apakah Go sudah terpasang dengan menjalankan perintah "go version".
   ```bash
   ~ go version
     go version go1.19.1 windows/amd64
   ```

## Tools untuk belajar golang ?

Berikut adalah beberapa alat yang dapat membantu Anda dalam mempelajari Go:

1. Go Playground: Ini adalah alat interaktif yang memungkinkan Anda menulis dan menjalankan kode Go tanpa mengatur lingkungan pengembangan Anda.

2. Go by Example: Situs web ini menyediakan contoh-contoh kode Go untuk membantu Anda memahami konsep-konsep tertentu.

3. Go Tutorials: Terdapat banyak tutorial online yang menjelaskan tentang Go dari dasar hingga tingkat lanjut.

4. Go CLI Tools: Beberapa alat CLI seperti Go dep, Go lint, dan Go fmt dapat membantu Anda dalam pengembangan aplikasi Go.

5. Text Editor atau IDE: VSCode, GoLand.

6. Roadmap Belajar Golang:https://roadmap.sh/golang
